# easy-wechat-gpt
The most easy CHAT-GPT Wechat bot

# How to run 
Open `bot.js`. Change L6 key to your own open ai key. 
Change L13 USER_NAME to your own wechat alias. 
Then 

```
node bot.js
```

# 如何运行
打开 `bot.js`. 将第六行换成你自己的OPEN-AI API KEY。 
将第十三行`USER_NAME`换成你自己的微信名。 

```
node bot.js
```
